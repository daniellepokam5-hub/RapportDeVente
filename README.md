Dashboard d’analyse des commandes (Power BI)

Ce projet présente un tableau de bord interactif réalisé avec Microsoft Power BI, permettant d’analyser les performances commerciales et les commandes annulées à travers plusieurs dimensions.
📌 1. Taux de commandes annulées par région
Visualisation sous forme de graphique en barres
Comparaison entre différentes régions (East, South, West, North)
Identification des zones avec le plus fort taux d’annulation
📌 2. Répartition du chiffre d’affaires par catégorie de produit
Visualisation en treemap
Analyse des catégories principales : Electronics, Accessories, etc.
Identification des produits les plus performants (Smartphones, Laptops, etc.)
📌 3. Évolution des commandes annulées dans le temps
Graphique temporel mensuel (janvier → décembre)
Suivi des tendances d’annulation
Comparaison par type de produit (Tablet, Smartphone, Headphones, etc.)
🎯 Objectifs du projet
Comprendre les causes et tendances des annulations de commandes
Identifier les régions et produits à risque
Optimiser les performances commerciales grâce à la data
Fournir un outil visuel d’aide à la décision
🛠️ Technologies utilisées
Microsoft Power BI
Modélisation de données
Visualisation interactive
Analyse de données
📈 Résultats clés
Mise en évidence des régions avec des taux élevés d’annulation
Identification des catégories générant le plus de revenus
Analyse des fluctuations mensuelles des annulations
